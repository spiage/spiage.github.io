    if-then-else¶

    Базовая условная конструкция в shell.

    Синтаксис¶

    if command; then
    # Выполняется, если command вернул 0
    echo "Success"
fi

    Полная форма:

    if command; then
    echo "True"
else
    echo "False"
fi

    Множественные условия:

    if [ "$a" -gt "$b" ]; then
    echo "a > b"
elif [ "$a" -lt "$b" ]; then
    echo "a < b"
else
    echo "a = b"
fi

    Условия¶

    Проверка команд¶

    if grep -q "pattern" file.txt; then
    echo "Pattern found"
fi

if mkdir /tmp/newdir; then
    echo "Directory created"
else
    echo "Failed to create directory"
fi

    Проверка с test / [ ]¶

    if [ -f file.txt ]; then
    echo "File exists"
fi

if [ "$name" = "Alice" ]; then
    echo "Hello Alice"
fi

    [[ ]] в bash¶

    Расширение bash с улучшенным синтаксисом:

    #!/bin/bash

# Логические операторы
if [[ -f file && -r file ]]; then
    echo "File exists and is readable"
fi

# Сравнение строк
if [[ "$name" > "Alice" ]]; then
    echo "Name comes after Alice"
fi

# Шаблоны
if [[ $file == *.txt ]]; then
    echo "Text file"
fi

    Преимущества [[ ]]: - ✅ &&, || вместо -a, -o - ✅ <, > без экранирования - ✅ Шаблоны == *.txt

    Примечание: [[ ]] — расширение bash. Для скриптов, которые должны запускаться в dash (или как #!/bin/sh), используйте классический [ ]. - ✅ Нет word splitting внутри

    Недостатки: - ❌ Не работает в dash/sh

    Практические примеры¶

    Проверка аргументов¶

    #!/bin/bash

if [ $# -eq 0 ]; then
    echo "Error: No arguments"
    echo "Usage: $0 <file>"
    exit 1
fi

if [ ! -f "$1" ]; then
    echo "Error: File not found: $1"
    exit 1
fi

echo "Processing $1..."

    Проверка переменных окружения¶

    #!/bin/bash

if [ -z "$DATABASE_URL" ]; then
    echo "Error: DATABASE_URL not set"
    exit 1
fi

if [[ "$DATABASE_URL" != postgresql://* ]]; then
    echo "Error: Only PostgreSQL supported"
    exit 1
fi

    Вложенные условия¶

    #!/bin/bash

if [ -d "$backup_dir" ]; then
    if [ -w "$backup_dir" ]; then
        echo "Backup directory ready"
    else
        echo "Error: Backup directory not writable"
        exit 1
    fi
else
    echo "Creating backup directory..."
    mkdir -p "$backup_dir"
fi

    Однострочные if¶

    # Короткая форма
[ -f file ] && echo "Exists"
[ -f file ] || echo "Not found"

# Эквивалентно
if [ -f file ]; then
    echo "Exists"
fi
